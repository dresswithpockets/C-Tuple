Detailed Description

Tuple.

An implementation of Tuples in C-Lang using void pointers.

Typedef Documentation

typedef struct tuple tuple

A Tuple Representation. A value of values.

A Tuple contains a collection of pointers to values, as well as information for if specific values were allocated on the stack or not.

If the flags for each t_ptr provided in t_stack are incorrectly assigned, memory leakage will occur.

See also
make_tuple(), free_tuple()
typedef struct tuple_ptr tuple_ptr

A Part of a Tuple.

A Tuple Pointer or a Tuple Part is a value that contains a pointer to a value and a flag suggesting whether or not the value was allocated on the stack.

If the on_stack flag is incorrectly applied, then memory leakage will occur.

See also
tptr(), tptr_stack(), free_tptr()
